Features

Oyster is a fully fledged election and motion system for any organisation.

Features for version 1.0 include:

User experience

  • A beautiful administration UI for creating and managing polls
  • A poll editor for easily creating and editing polls
  • An easy to use, themable ballot interface for end users
  • Extremely readable and understandable results page automatically generated when a poll concludes

Polls

  • Handle motions with configurable thresholds (simple majority, two-thirds, etc)
  • Support the following election types:
    • Schulze method Condorcet voting
    • Approval voting
    • Score voting
  • Automatically scheduling of start and end of voting periods
  • Automatic generation of results

Privacy and integrity

  • Unique ballots delinked from a participant's identity
  • Fully published data set at the conclusion of an election for verification purposes
  • Full audit logs for administrators

Internationalisation

  • Fully localisable admin interface, results output and poll data

Licensing

  • Completely free and open source under the Affero General Public License version 3

And that's just the beginning!

Get involved, take a look at the code and get excited for version 1.0, due mid-November 2015.

A roadmap will be provided upon release.
