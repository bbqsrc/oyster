Oyster A free open-source voting system that you can host yourself.

Oyster is a web-based voting system that can be used for handling motions and elections within your organisation securely and with ease.

Unique, anonymous tokens are sent to each participant's email address. The link takes a participant to a form that can be completed from any mobile or desktop device, and can verify that the vote has been added to the system both during and at the conclusion of the voting period.

Check out the features page for the details of what it can do.

Development

Development is currently approaching version 1.0. It is scheduled for release in mid-November, 2015.
